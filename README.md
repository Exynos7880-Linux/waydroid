# waydroid installer script for sailfishos Exynos7880 devices

## Usage

1. install git  
```
devel-su zypper in git
```
2. clone the repo
```
git clone https://github.com/Exynos7880-Linux/waydroid.git --recurse-submodules --depth 1
```
3. cd waydroid
```
cd waydroid
```
4. install waydroid 
```
devel-su ./waydroid.sh vanilla|gapps
```
